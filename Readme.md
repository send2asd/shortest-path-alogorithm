# Min-hop  path  finding  problem

#
To Run this python script you can use the Below mentioned command

```python3 main.py <no. of nodes> <no. of edges>```
##

##
The output will be generated in output.txt which will include 
> the start and end node and the path in form of a list and the time taken in micro seconds
_example:-_

```
Min-hop between 26 and 6:[26,76,6]
Time taken in microsecond : 163
```

#
To Run the whole code at once user can use shell script i.e.

``` bash bash.sh```
